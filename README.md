# Team Guidelines

Guidelines For Front-End Developers.

团队指引，包括学习指南、规范文档等。

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/elephant-fe/guide
$ cd guide
$ npm install
```

Generate:

``` bash
$ hexo g
```

Run server:

``` bash
$ hexo s --watch
```

## Deployment

1. Generate and optimize assets

  ```bash
  gulp
  ```

2. Deploy to the gh-pages branch

  ```bash
  hexo d -g
  ```
